மொட்ட பையன்


பெண் : மொட்ட பையன்…. மொட்ட பையன்….
ரொம்ப ரொம்ப கெட்ட பையன்….

பெண் : மொட்ட பையன்…. மொட்ட பையன்….
ரொம்ப ரொம்ப கெட்ட பையன்
கெட்ட பையன்…. கெட்ட பையன்…..
என் உசுர தொட்ட பையன்

ஆண் : இந்த உறவு கெடைக்குமா
நெஞ்சம் தவிச்சதே
அந்த வரமும் இன்றுதான்….
கையில் கெடச்சதே

பெண் : நம் காதல்
ஜெய்க்கும் நேரமே…
கண்ணில் கடவுள் தோன்றுமே….

ஆண் : உன் பாத கொலுசு…
மோதித் தெறிக்கும்
நானும் ரசித்து
நாளைக் களிப்பேனே

பெண் : மொட்ட பையன்…. மொட்ட பையன்….
ரொம்ப ரொம்ப கெட்ட பையன்

பெண் : நெஞ்சுக்குள்ளே இருந்தது
வார்த்தைகளாய் விழுந்தது
இந்த நொடி இந்த நொடி
மண்ணிலே சொர்க்கம் கண்டேன்
என்னவனை தந்ததற்கு
சாமிக்கு நன்றி சொன்னேன்

பெண் : ஒரு கோடி வருஷம்
வாழ்ந்தான பிறகும்
என் உடல் அதிர உயிர் அதிர
உன்னை நினைத்து என்னை மறந்தேனே

ஆண் : உன்னைவிட அழகிகள்
எத்தனையோ உலகிலே
இருக்கலாம் அது எல்லாம்
எனக்கழகு இல்லை

ஆண் : சிரிக்கும் உன் சிரிப்புதான்
அழகுகளின் எல்லை
தேரோடும் தெருவில்
நீ போக பார்த்தால்
நான் மதம் மறந்து
உனதருகில் இடம் பிடிக்க
வந்து தவிப்பேனே

பெண் : மொட்ட பையன்…. மொட்ட பையன்….
ரொம்ப ரொம்ப கெட்ட பையன்

ஆண் : இந்த உறவு கெடைக்குமா
நெஞ்சம் தவிச்சதே
அந்த வரமும் இன்றுதான்….
கைய ில் கெடச்சதே

பெண் : நம் காதல்
ஜெய்க்கும் நேரமே…
கண்ணில் கடவுள் தோன்றுமே….

ஆண் : உன் பாத கொலுசு…
மோதித் தெறிக்கும்
நானும் ரசித்து
நாளைக் களிப்பேனே

பெண் : ஹ்ம்ம்……
மொட்ட பையன்…. மொட்ட பையன்….
ரொம்ப ரொம்ப கெட்ட பையன்….
